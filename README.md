# Caleb Mumo Kisangau | Portfolio - SkySoft Solutions

Welcome to my modernized 2025 portfolio project. Organized into parts for maximum scalability and clean architecture.

## 📁 Directory Structure

- `index.html`, `contact.html`, `thanks.html` : The main page "shells".
- `public/`: Assets like images and uploads.
- `src/layout/`: HTML modular components like `header.html` and `footer.html`.
- `src/pages/`: Content for individual pages.
- `src/css/`: Global styles and modern theme variables.
- `src/js/`: Client-side logic and modular component loader.
- `src/experiment/`: Experimental layouts and UI fragments.

## 🚀 Getting Started

To run the project with modular component loading (using `fetch`), you need a simple HTTP server. 

You can use:
- **VS Code Live Server**
- `npm run dev` (if you have Node.js)
- `python -m http.server 8000`

## 💎 Features

- **Modular Design**: Header and Footer are shared across all pages.
- **Glassmorphism**: A sleek, modern aesthetic with translucent glass effects.
- **Micro-animations**: Smooth reveal-on-scroll effects.
- **Performance**: Optimized lazy loading for images and fonts.
- **Accessibility**: ARIA-labeled buttons and semantic layout.

## 🛠️ Built With

- **Tailwind CSS** (for styling)
- **Vanilla JavaScript** (for logic and components)
- **FontAwesome** (for iconography)
- **Google Fonts** (Outfit/Inter)
